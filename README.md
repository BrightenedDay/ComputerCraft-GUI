# Installation

Paste into computer.
```
wget https://raw.githubusercontent.com/BrightenedDay/ComputerCraft-GUI/master/gui.lua gui
```
And then insert this line into your program.
```lua
local gui = require("gui")
```
And done! You've installed it!
